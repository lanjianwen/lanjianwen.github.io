利用Github+Hexo搭建个人博客

准备工作

  1. 申请github账号
    github官网

  2. 下载并安装 Node.js
    下载与安装教程

  3. 下载并安装 Git
    下载与安装教程

  4. 安装Hexo和本地部署博客
    4.1. 新建文件夹Hexo(以后代码都放这)
    4.2. 在 Hexo 目录下 鼠标右键选择Git Bash Here
    4.3. 输入命令npm install -g hexo-cli
    4.4. 查看是否安装成功，输入hexo -v
    安装成功
    4.5. 输入hexo init回车->输入npm install之后会生成一些文件夹
    4.6. 输入hexo s->打开浏览器输入http://localhost:4000/即可看到本地博客部署

    github搭建github博客

  5. 登陆自己的github账号

  6. 新建一个仓库 ：点击在右上角+->New repository->在Repository name 输入仓库名字 xxx.github.io以后仓库名就是你的博客地址。

  7. 配置SSH Key ：一般在你的C盘用户目录下->找到隐藏文件.ssh文件夹->有私钥id_rsa 公钥id_rsa.pub->复制公钥(如果没有.ssh文件夹，打开Git Bush输入 ssh-keygen -t rsa -C "你的邮箱地址")在你的github选择settings进入到

  8. 在Git Bush输入ssh -T git@github.com查看是否配置成功

    将本地博客部署到github

  9. 安装github插件依赖：在Hexo文件目录下打开Git Bash输入npm install hexo-deployer-git --save

  10. 修改站点文件(Hexo文件夹下_config.yml)

  11. 输入hexo g && hexo d->在浏览器输入http://xxx.github.io即可看到本地博客部署到github上s
